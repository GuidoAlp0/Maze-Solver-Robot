This project implements an autonomous maze-solving robot using the Left Wall Follower algorithm. 

The project unites embedded programming, sensor integration, and autonomous navigation to demonstrate a simple yet effective maze-solving strategy.

## Features

* Left Wall Follower navigation algorithm
* Autonomous obstacle detection
* Real-time decision making at intersections
* Motor control for autonomous movement
* Embedded implementation for mobile robotics

## Hardware

* PIC18F16041
* Distance sensors
* DC motors with motor driver
* Mobile robot chassis
* Battery power supply

## Software

* Embedded C
* Autonomous navigation state machine
* Sensor-based decision logic
* Motor control routines
